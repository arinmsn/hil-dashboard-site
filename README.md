# HiL Test Automation Dashboard

A Python desktop application built for hardware-in-the-loop (HiL) test automation. The dashboard provides a unified GUI for configuring test suites, executing Pytest runs, monitoring live output, and exporting results — without touching the terminal.

Built as a Senior Capstone Project for Computer Science.

---

## Features

- Checkbox-based test suite selection and configuration
- One-click Pytest execution via subprocess
- Real-time log streaming during test runs
- Key Performance Indicators (KPI) summary cards (pass/fail/skip counts)
- Export results to CSV and PDF
- Modular architecture separating GUI from execution logic

---

## Tech Stack

| Layer      | Technology              |
|------------|-------------------------|
| GUI        | Python, Tkinter          |
| Testing    | Pytest, subprocess       |
| Reporting  | CSV, PDF export          |
| Version Control | Git, GitHub         |
