Test Suite Configuration

Users select which test modules to run using the checkbox panel on the left side of the dashboard. Parameters such as timeout and verbosity can be adjusted before execution begins.

📷 Screenshot placeholder — Test Suite Selection Panel

Live Execution Log

Once the Run button is clicked, Pytest output streams directly into the log panel in real time. Each test case result appears as it completes, color-coded by status.

📷 Screenshot placeholder — Live Execution Log (Tests Running)

KPI Summary & Results View

After execution completes, the KPI cards update to reflect the final pass, fail, and skip counts. A summary report can then be exported directly from the dashboard.
